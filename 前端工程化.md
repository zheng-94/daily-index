# 前端工程化
试图小结已有框架解决的问题：
- JQ 解决DOM 操作，浏览器兼容性
- Backbone 解决数据与视图的耦合
- React / Vue / Angular 解决 DOM 操作及数据与视图的耦合
- Apollo / Relay 解决 API 与数据的割据
- RN / Flutter / Taro / Uniapp 解决安卓苹果小程序的分裂

- 微前端、monorepo 解决大型项目弱管理
- serverless、nest/next/nuxt 解决前后端联调难题
